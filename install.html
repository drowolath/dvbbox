Installation¶

The recommended way is to go

$ pip install dvbbox

Install from source¶

You can download the source and use the setup.py script to install:

$ git clone https://github.com/drowolath/dvbbox.git
$ cd dvbbox
/dvbbox$ python setup.py install

Install from .deb¶

The application comes also packaged as a .deb file located here . You can choose the one corresponding to your architecture and do:

$ sudo dpkg -i <the_package>
