.. _dvbbox_install:

Installation
============

The recommended way is to go

.. code-block:: bash

   $ pip install dvbbox

Install from source
-------------------

You can download the source and use the setup.py script to install:

.. code-block:: bash

   $ git clone https://github.com/drowolath/dvbbox.git
   $ cd dvbbox
   /dvbbox$ python setup.py install

Install from .deb
-----------------

The application comes also packaged as a .deb file located `here <https://github.com/drowolath/dvbbox/tree/master/deb/>`_ . You can choose the one corresponding to your architecture and do:

.. code-block:: bash

   $ sudo dpkg -i <the_package>
